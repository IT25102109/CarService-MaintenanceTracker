SE1020 — Object Oriented Programming | Group Project

## Team Members
- Member 1 — User Management
- Member 2 — Booking Management
- Member 3 — Staff & Mechanics
- Member 4 — Parts & Inventory
- Member 5 — Service Records
- Member 6 — Reviews & Notifications

## Tech Stack
- Java JDK 25 (Amazon Corretto)
- Apache Tomcat 11
- JSP + JSTL
- MySQL via MAMP
- Tailwind CSS

## How to Run
1. Clone the repo
2. Open in IntelliJ IDEA
3. Start MAMP and create `car_service_db`
4. Run `sql/schema.sql` in phpMyAdmin
5. Configure Tomcat and run
